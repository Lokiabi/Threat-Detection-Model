**📌 Project Overview**

This project focuses on building a network intrusion detection system (IDS) using a Random Forest machine learning model. The system analyzes network traffic and classifies it as normal or malicious (attack).

The model is trained using the UNSW-NB15 dataset, a widely used dataset for cybersecurity research that contains modern network attack scenarios.

The final trained model is exported as a .pkl file, making it easy to deploy in applications or integrate into security monitoring systems.

**🎯 Objectives**

Detect malicious network traffic using machine learning

Classify network packets into normal or attack categories

Build a reliable and scalable intrusion detection model

Export the trained model for real-world deployment

**📊 Dataset**

The model is trained on the UNSW-NB15 dataset, which contains:

Modern network traffic features

Multiple attack categories

Realistic normal and malicious network behavior

Features include:

Network flow statistics

Protocol information

Packet-level attributes

Traffic behavior metrics
